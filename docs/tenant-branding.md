# Personalización por Academy

Cada organización puede configurar su experiencia sin cambiar la identidad base de Academy by Scalaria:

- Nombre visible.
- Logo por URL.
- Color principal.
- Mensaje de bienvenida.
- Módulos visibles en el portal.
- Dominio personalizado preparado para la siguiente fase de hosting.

La configuración se guarda en `AcademyOrganization` y se aplica al portal autenticado del tenant activo.
