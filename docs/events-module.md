# Eventos y asistencia

Academy permite que los alumnos se registren a eventos y webinars publicados por su organización. El registro queda asociado al usuario, evento y organización.

Administración puede consultar la lista de inscritos y alternar el estado entre registrado y asistió. Resend envía confirmaciones automáticas y permite mandar recordatorios desde el panel, usando el dominio verificado `scalaria.com.mx`. Los enlaces de acceso y grabaciones quedan como la siguiente iteración operativa.
